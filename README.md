# The Hidden Engine of LEGO Growth

## Project Overview
This project is a Power BI data storytelling report created for the LEGO visualization challenge using data from the Rebrickable LEGO Database.

The report explores a less obvious idea about the LEGO ecosystem: LEGO's growth is not only a story of more sets, more themes, or more pieces. The deeper pattern is that LEGO creates huge variety while still relying on a relatively stable core of colors, part categories, and design logic.

## Key Takeaway
LEGO sets became much larger over time, but most builds still rely on a small core of colors and familiar part types. The surprise is that LEGO scales creativity through constraint, not constant reinvention.

## Dashboard Pages

### Page 1: The Hidden Engine of LEGO Growth
This landing page introduces the main story of the report. It presents the key takeaway, a small set of headline KPIs, and a trend chart showing that LEGO sets have grown much larger over time while color diversity has increased more slowly.

![Page 1 - The Hidden Engine of LEGO Growth](images/LegoGrow_page1.png)

### Page 2: A Small Palette Powers Huge Variety
This page provides the strongest evidence for the story. It shows that a small set of colors accounts for most LEGO part volume, and that a few part categories carry much of the system. The page supports the idea that LEGO creates variety through recombination rather than endless expansion.

![Page 2 - A Small Palette Powers Huge Variety](images/PalettePower_page2.png)

### Page 3: Some Themes Tell Stories with Characters, Others with Structure
This page adds a human angle to the story through minifigs and theme differences. It compares themes by how character-driven they are and highlights that some sets create richness through narrative play, while others rely more on scale, form, or design complexity.

![Page 3 - Some Themes Tell Stories with Characters, Others with Structure](images/TellStories_page3.png)

## Accessibility and Design Notes
The report was designed with a storytelling-first structure:
- concise page narratives instead of chart-heavy layouts
- clear chart titles and annotations
- consistent color palette
- readable typography
- visual hierarchy that supports quick understanding


This report is designed as a short analytical story rather than a traditional dashboard. The goal is to help viewers leave with one memorable insight: LEGO's creativity grows through constraint.
